MOKU OFFLINE GUIDE

使用说明

这份说明随 MOKU 离线保存。打开和阅读不会连接 Pixiv；只有点击“检测网络”才会发起匿名连通测试。

快速开始

输入标签后点击“开始寻找”。多个标签用空格分开，例如“猫 夜景”，表示分别搜索后按作品 ID 合并去重（OR），翻页仍保持完整标签组合。

网络连接

开启 Windows 系统代理时，MOKU 使用该电脑当前启用的本机 HTTP 代理；开启 TUN 全局模式时，MOKU 的直连请求由系统路由接管。两者都关闭且无法直连 Pixiv 时，搜索会失败。

检测只匿名访问 Pixiv 主站和图片线路，不发送 Cookie；不会修改 Windows 系统代理，不会自动启动 VPN，也不会扫描本机端口。

尚未检测当前路线：待检测

遇到连接问题时，可在这里检查当前系统代理或 TUN 路径是否能访问 Pixiv。

Pixiv 主站：待检测图片线路：待检测

登录与隐私

密码、验证码和 2FA 只在 Pixiv 官方页面输入。选择保持登录时，仅保存严格筛选后的会话到 Windows 凭据管理器；网页端不会得到 Cookie。不支持 R-18G。

搜索与分页

每页显示 36 件作品，并预加载后续 3 页的作品数据；不会提前下载未打开页面的缩略图。向后翻页时保留前 6 页，超出窗口的旧页数据和临时预览授权会清理。

下载与文件

默认保存到便携目录的 downloads 文件夹，也可选择其他绝对路径。多页作品可逐页勾选；请尊重作者版权、作品用途和 Pixiv 平台规则。

SHA-256

SHA-256 是 MOKU.exe 的单向文件指纹，不包含账号、Cookie、路径或身份信息。可用 Get-FileHash .\MOKU.exe -Algorithm SHA256 与 SHA256.txt 核对。

常见问题

主站可用但图片失败：代理可能没有覆盖 i.pximg.net。
复制到其他电脑：请复制完整 MOKU 文件夹，并在目标电脑安装 WebView2 Runtime。
系统代理关闭：若未开启 TUN 且不能直连 Pixiv，MOKU 也无法连接。

LOCAL ACCOUNT CONNECTION

连接 Pixiv

为了账户安全，本应用不会要求你在此输入 Pixiv 密码，也不会把 Cookie 或 Token 保存到网页存储。

当前：未授权将在 MOKU 桌面登录窗口打开 Pixiv 官方页面；登录过程中会实时监控授权状态。
